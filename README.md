# Portfólio Pessoal - Rafael S. Figueiredo

Bem-vindo ao repositório do meu portfólio pessoal! Este projeto é a minha casa na web, uma vitrine digital onde compartilho minhas habilidades profissionais, experiências na área da tecnologia, e os principais projetos pessoais e profissionais que já desenvolvi.

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando um ecossistema moderno visando entregar alta performance, SEO e um design altamente responsivo, sofisticado e rápido:

- **[Next.js](https://nextjs.org/)** (v16) - Framework robusto voltado para produção
- **[React](https://react.dev/)** - Componentização e interfaces dinâmicas
- **[Tailwind CSS](https://tailwindcss.com/)** (v4) - Framework CSS focado em utilitários para construir meu Design System focado em Dark Mode com Glassmorphism.
- **[Framer Motion](https://www.framer.com/motion/)** - Biblioteca mágica para transições limpas, scroll reveal interativo e micro-animações.
- **[TypeScript](https://www.typescriptlang.org/)** - Segurança e tipagem em todo o código.

## 🌟 Funcionalidades

- **Design Premium UI**: Baseado em um moderno tema dark com paleta indigo e violeta sutil, aurora customizada no background, botões dinâmicos, glassmorphism e overlay filter nas fotos.
- **Animações Fluídas**: Micro-interações agradáveis em todas as seções por conta do framer-motion (cards que flutuam, Typewriter effect dinâmico e scroll spy para ver qual página está rolando no momento).
- **Experiência Perfeita**: Layout flexível, preparado tanto para telas Ultra Wide quanto para pequenos celulares.
- **Contatos integrados**: Funcionalidade real para que eu receba contatos, incluindo opções para baixar currículo localmente ou redirecionar o visitante diretamente ao meu e-mail ou WhatsApp.

## 🛠️ Como rodar o projeto localmente

Caso deseje clonar meu código para inspecionar localmente (ou rodar na sua máquina), execute os seguintes passos no seu terminal:

```bash
# Clone este repositório
git clone https://github.com/RaelSF/portfolio.git

# Acesse a pasta raiz do projeto
cd portfolio

# Instale os pacotes e as dependências listadas
npm install

# Execute a aplicação em um ambiente local de desenvolvimento
npm run dev
```

A aplicação fará o start-up e as mensagens no terminal informarão o seu localhost. Você poderá visualizar diretamente no seu navegador, abrindo `http://localhost:3000`.
